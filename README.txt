A lightweight Maintenance Mode plugin for WordPress that displays a custom page for non-logged-in users.

== Description ==

**Ace Maintenance Page** allows you to quickly put your WordPress site into maintenance mode, displaying a custom message, logo, background, and colors for visitors while still allowing administrators to view the site.

Features include:

* Customizable maintenance page title and description
* Upload your logo and set a background image/color
* Exclude certain pages from maintenance mode
* Mobile responsive
* Admin preview before activating
* Fully secure and WordPress.org review friendly

This plugin is perfect for website owners who need to temporarily take the site offline while performing updates, redesigns, or maintenance.

== Installation ==

1. Upload the `ace-maintenance-page` folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the **Plugins** menu in WordPress.
3. Go to **Settings â†’ Ace Maintenance Page**.
4. Enable maintenance mode and customize your page.
5. Save changes. Non-logged-in users will see the maintenance page.

== Frequently Asked Questions ==

= Can I preview the maintenance page before enabling it? =

Yes. Admins can preview the maintenance page using the preview link in the settings.

= Can I exclude pages from maintenance mode? =

Yes. You can enter page slugs in the "Exclude Pages" option.

= Will logged-in users still see the site? =

Yes. Logged-in users with admin permissions see the normal site while maintenance mode is active.
